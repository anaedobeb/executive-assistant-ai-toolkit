# Executive Assistant AI Toolkit

A collection of AI-assisted workflows, systems, and operational templates designed to support fast-moving executives and remote teams.

## Purpose

This repository explores how AI tools such as ChatGPT and Claude can improve executive support, communication, scheduling, and operational efficiency.

## Areas Covered

* Executive calendar coordination
* AI-assisted meeting summaries
* Travel planning systems
* Follow-up management
* Time zone coordination
* Productivity workflows
* Administrative systems
* Communication templates

## Tools

* ChatGPT
* Claude
* Notion
* Google Workspace
* Canva
* Slack
* Zoom

## Current Focus

Building operational systems that reduce repetitive work and improve communication quality in remote-first environments.
